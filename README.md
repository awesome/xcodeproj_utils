# XcodeprojUtils

Count source lines of files in xcode project.

## Installation

Add this line to your application's Gemfile:

    gem 'xcodeproj_utils'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install xcodeproj_utils

## Usage

	$ xcp_utils lines <PATH_TO_XCODE_PROJECT> <TARGET_NAME>
